# Feature Summary — quick reference


> "Physical Reading Mode brings the soul of real books to Kindle — pages wrinkle when dropped, yellow with age, and rustle when turned. It's data-driven nostalgia that makes students read 15% longer."

---

## Feature → data signal → metric

| Feature | Input data | Output metric to track |
|---------|-----------|----------------------|
| Drop wrinkling | Accelerometer impacts | Engagement delta (A/B) |
| Page yellowing | Timestamp + reading hours | Session duration |
| Ink bleed-through | Annotation coordinates + stylus pressure | Annotation frequency |
| Page-turn sound | Swipe velocity + page ratio | Pages per session |
| Thickness bar | Page count + position | Navigation pattern changes |
| Gravity flip | Gyroscope orientation | Reading position insights |

---

## Key numbers to remember

- **MVP timeline:** 3 weeks (sound + yellowing)
- **North star:** Weekly active reading minutes, +15% target
- **Adoption target:** 40% of student users enable within 30 days
- **Session duration target:** +10% vs. control
- **Retention target:** +8% 30-day return rate
